# 🚀 ClassTeams — Smart Academic Dashboard (VERS HTML)

> Built for students who want control, not chaos. 🎯

---

## 🧠 What is ClassTeams?

**ClassTeams** is a modern academic dashboard inspired by Google Classroom —
now powered with a real backend using Supabase.

This isn’t just UI anymore.
This is a **full-stack learning platform prototype**.

---

## ⚡ Tech Stack

* ⚛️ Next.js (App Router)
* 💙 TypeScript
* 🎨 Tailwind CSS
* 🗄️ Supabase (Database + Auth ready)

---

## 🔥 Features

* 🧭 Sidebar Navigation (active route detection)
* 📝 Assignment & Post UI
* 📅 Schedule Management
* 📂 Resource Section
* 💬 Discussion System (UI + ready for backend)
* 🔌 Supabase integration (database ready)

---

## 🗄️ Database (Supabase)

Powered by **Supabase** as Backend-as-a-Service:

* PostgreSQL Database
* RESTful API auto-generated
* Authentication (ready for future)
* Real-time capabilities (optional upgrade)

## ⚙️ Environment Setup

Create `.env.local` file:

```bash
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
```

Get these from your Supabase project dashboard.
